Du er en ekspert webutvikler og pedagogisk matematikklærer.

Lag en komplett nettside som genererer matteoppgaver for barn.

STRUKTUR:
Løsningen skal bestå av:

1. index.html
2. styles.css
3. js/script.js (entrypoint)
4. støttemoduler under js/

Alle filene skal være komplette, ryddige og klare til bruk.

------------------------------------

FUNKSJONALITET:

1. Brukeren skal kunne velge:
   - Alder: 10 år eller 12 år
   - Antall matteark
   - Vanskelighetsgrad:
        * Lett
        * Middels
        * Vanskelig
        * Blandet

2. Nettsiden skal generere matteark med denne strukturen:

A. Addisjon og subtraksjon (5 oppgaver)
B. Multiplikasjon og divisjon (5 oppgaver)
C. Brøk og prosent (5 oppgaver)
D. Geometri (5 oppgaver)
E. Trigonometri (5 oppgaver)
F. Tekstoppgaver (5 oppgaver)

3. Oppgavene skal:
   - Være varierte (ingen gjentakelser)
   - Ha stigende vanskelighetsgrad innen hvert ark
   - Passe til valgt alder
   - Inneholde realistiske tekstoppgaver
   - Ha korrekt fasit (uten feil)

------------------------------------

GEOMETRI:
- Areal, omkrets og volum
- Pytagoras
- Vinkler
- Tegn figurer med SVG (rektangler, trekanter osv.)

------------------------------------

TRIGONOMETRI:
- Bruk sinus, cosinus og tangens
- Rettvinklede trekanter
- Finn sider og vinkler
- Vis figurer med SVG

------------------------------------

UI / FUNKSJONER:

- Knapp: "Generer oppgaver"
- Knapp: "Vis fasit" (vis/skjul)
- Knapp: "Skriv ut" (A4 optimalisert)

- Vis matteark pent strukturert
- God luft mellom oppgaver
- Barnevennlig design

------------------------------------

PRINT:

- CSS print-oppsett:
   - A4 format
   - Skjul knapper
   - Pen layout for utskrift

------------------------------------

TEKNISKE KRAV:

- Bruk ren HTML, CSS og JavaScript (ingen rammeverk)
- Bruk modulstruktur i JS (funksjoner delt opp logisk)
- Koden skal være godt kommentert
- Bruk MathJax via CDN for matematikkvisning

------------------------------------

FILKRAV:

index.html:
- Struktur, input-felt, knapper
- Kobling til CSS og JS
- MathJax import

styles.css:
- Rent og profesjonelt design
- Barnevennlig fargevalg
- Print-stiler

js/script.js:
- Generering av oppgaver
- Generering av fasit
- Random tall
- SVG-figurer for geometri og trigonometri
- Event handlers for knapper

------------------------------------

EKSTRA (VALGFRITT MEN BRA):

- Highlight vanskelige oppgaver
- Mulighet for å vise/skjule figurer
- Del opp i flere matteark på siden
- Sørg for at alt fungerer uten feil

------------------------------------

OUTPUT:

Returner KUN:

1. index.html
2. styles.css
3. js/script.js

Ved modulstruktur: inkluder nødvendige filer under js/.

Alle fullstendige og klare til bruk.
